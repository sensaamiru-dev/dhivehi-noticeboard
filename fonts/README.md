# Fonts Directory

Place your Dhivehi font files here.

## Supported Fonts

### 1. MV Waheed
- File names: `MV_Waheed.otf`, `MV_Waheed.ttf`, or `A_Waheed.ttf`
- Download from: https://www.thaanafontgallery.com/font/mv-a-waheed/
- Traditional Maldivian font

### 2. Faruma
- File names: `Faruma.otf`, `Faruma.ttf`, or `A_Faruma.ttf`
- Download from: https://raajjefonts.github.io/
- Modern Maldivian font

## How to Add Fonts

1. Download the font files (.otf or .ttf format)
2. Copy them to this `/fonts` folder
3. Refresh the webpage
4. Use the font toggle buttons on the left side to switch fonts

## Default Font

If no local fonts are installed, the app will use **Noto Sans Thaana** from Google Fonts automatically.

## Font Licensing

Please ensure you have the right to use these fonts. Most Dhivehi fonts are free for personal and commercial use, but always check the specific license.
